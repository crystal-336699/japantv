🔧 VLC 웹인터페이스 설정

최초 1회만 하면 돼요

  1. VLC 실행
  2. 도구 → 환경설정
  3. 왼쪽 하단 "모두 보기" 클릭
  4. 인터페이스 → 메인 인터페이스 → "웹" 체크
  5. 인터페이스 → 메인 인터페이스 → Lua
    → HTTP 비밀번호에 1234 입력
  6. 저장 후 VLC 재시작
